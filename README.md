# vyand

Vyand is a wifi packet sniffer

```
$ vyand --help

Usage: vyand [OPTIONS]

  monitor wlan

Options:
  -i, --interface TEXT   The wireless LAN interface
  -c, --channel INTEGER  The wireless LAN interface  [default: 1]
  --help                 Show this message and exit.

```
